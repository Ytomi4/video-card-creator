## UI 改善項目

### プレビューの単一化

- プレビューは合成プレビューのみにして、中央に配置
- フレームを選んだ時、また動画を読み込んだ時、それぞれひとつの canvas に反映されて確認できるようにしてほしい

### 動画のインポートの簡便化

- ドラッグ＆ドロップで動画を読み込めるようにしたい

### 視覚的な改善

- Figma のデザインシステムに準拠
- 重要なアクションの協調（録画ボタン）
- 直感的なアイコンの仕様
